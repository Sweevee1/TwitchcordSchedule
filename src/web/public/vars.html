TwitchcordSchedule

Template Variables

Use variables in your event title and description templates to automatically fill in live details from the Twitch schedule. Variables are written as {variable_name} and replaced at sync time.

Available variables
Variable Description Available in Example value
{title} The scheduled stream title set by the broadcaster on Twitch. Falls back to an empty string if not set.
Title Description
Ranked Games
{category} The game or category the stream is scheduled under. Empty string if no category is set for the segment.
Title Description
Valorant
{broadcaster} The broadcaster's display name as it appears on their Twitch channel.
Title Description
Sweevee
Example
Title template
{broadcaster} — {title}
Description template
Tune in for some {category}! Follow at twitch.tv/{broadcaster}
↓ after sync
Event title
Sweevee — Ranked Games
Event description
Tune in for some Valorant! Follow at twitch.tv/Sweevee
Unknown variables — any {variable} that doesn't match a known name is replaced with an empty string. Capitalisation matters: {Title} will not match {title}.
